# 闭包

‘闭包’，顾名思义：内部的一个包 

- js中所有的函数都是闭包，因为至少都持有全局环境

- 当代码试图访问一个变量时 —— 它首先会在内部词法环境中进行搜索，然后是外部环境，然后是更外部的环境，直到（词法环境）链的末尾。

- 闭包成就了从外部访问内部变量。（实现了私有变量）

- 函数currying

* 闭包滥用会造成内存泄漏

```javascript
function f() {
  let i = 1;
  return function g () {
    alert(i)
  }
}
let g = f(); // g 是可达的，并将其外部词法环境保持在内存中
```
