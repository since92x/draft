# 三次握手

首先： ACK = Seq + 1

1. client 发送(SYN = 1, Seq = X, ACK = 0)
2. server 发送(SYN = 1, Seq = Y, ACK = X + 1)
3. client 发送(SYN = 1, ACK = Y + 1)

# 四次挥手
退订发送Fin + ACk
确认发送Ack + Seq
1. client 发送(Fin = 1, Ack = Z, Seq = X)
2. server 发送(Ack = X + 1, Seq = Z)
3. server 发送(Fin = 1, Ack = X, Seq = Y)
4. client 发送(Ack = Y, Seq = X)
